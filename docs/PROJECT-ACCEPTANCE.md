# Project Acceptance

项目只有在本文件所有高优先级验收项都通过后，才能视为完成。

## 全局完成定义

- 所有 `PROJECT-TODO.md` 中的任务都已完成或被明确废弃并记录决策
- 所有阶段退出条件满足
- 所有高优先级验收项通过
- 文档与代码状态一致
- 原始需求文档中的关键能力全部有实现或经过已记录的降级说明
- 若 git 仓库和远端已就绪，最终结果已提交并推送
- 若 git 条件尚未就绪，阻塞已被明确记录且等待用户完成仓库初始化
