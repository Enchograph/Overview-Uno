- 当前登录态本地存储为客户端本地 JSON 文件，尚未接入平台级安全存储；首版先满足登录态恢复，后续若进入平台安全加固需在 Platform/QA 阶段补齐
- 当前 `IUserSettingsService.GetAsync` 在本地无记录时返回默认设置对象，但不会立即持久化；真正写入与同步登记发生在 `SaveAsync`
